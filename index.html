Timetable Studio

A clean monochrome timetable builder for courses, labs, tutorials, practicals, work blocks, and weekly planning.

How course toggles work:
A course can be visible, shadowed, or hidden. Visible courses count in the schedule; shadow courses show faintly but do not count; hidden courses disappear entirely. Multiple tutorial/practical options stay tucked inside each course accordion.
